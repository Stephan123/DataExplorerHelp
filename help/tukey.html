Definition
Graphical technique for identifying the shape parameter for a distributional family that best describes the dataset.
The Tukey lambda distribution has for quantile `Q(p) = (p^lambda-(1-p)^lambda)/lambda`.
We do a series of probability plot of the data with the Tukey lambda distribution for λ between -1 and 1, and we plot the correlations. The maximum of the curve gives the best value of the shape parameter:
λ = -1Approximately Cauchy C(0,π)
λ = 0Exactly logistic
λ = 0.14Approximately normal N(0,2.142)
λ = 0.5Strictly concave
λ = 1Exactly uniform U(-1,1)

Parameters
In the top slot, the qualitative field whose categories are to be represented.
