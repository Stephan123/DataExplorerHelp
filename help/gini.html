Definition
Maß dafür, wie oft ein zufällig ausgewähltes Element falsch gelabelt werden würde, wenn man das Label zufällig aus der Verteilung der Kategorien auswählt.
Parameter
Oberes Feld: kategoriale Variable, die analysiert werden soll.
Linkes Feld: optionale kategoriale Variable. Der Gini-Index wird für jede Kategorie dieser Variable berechnet.
