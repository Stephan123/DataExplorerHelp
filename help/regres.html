Definition
Berechnet den Zusammenhang zwischen abhängiger und unabhängigen Variablen.
Parameter
Rechte Felder: quantitative Variablen, die als unabhängige (erklärende) Variablen verwendet werden sollen.
Linkes Feld: quantitative abhängige Variable.
WARNUNG: Im Fall einer logistischen Regression mit quantitativer Response-Variable wird die angegebene Variable als Anzahl der Erfolge genutzt, das nächste Feld als Anzahl der Misserfolge.
Optionen
Option 1: Zusammenfassung der Regression.
Option 2: Die Koeffizienten.
Option 3: Weitere Resultate:
  1. Gefittete Werte
  2. Vorhergesagte Werte
  3. Deviance-Residuen
  4. Pearson-Residuen
  5. Leverage (hat values)
  6. Cook's Distanz

Aktionen
Ziehen Sie den ersten blauen Pfeil auf die Liste quantitativer Variablen, um eine neue quantitative Variablen aus den vorhergesagten Werten zu erstellen.
Ziehen Sie den zweiten blauen Pfeil auf die Liste quantitativer Variablen, um eine neue quantitative Variable aus den Fehlern zu erstellen (Abweichungen der vorhergesagten von den wahren Werten).
In Option 3: Ziehen Sie den blauen Pfeil auf die Liste quantitativer Variablen, um eine neue quantitative Variable zu erstellen.
