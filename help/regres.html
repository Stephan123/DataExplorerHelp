Definition
Compute the relationship between the dependent variable and the explanatory variables.
Parameters
In the right slots, the numerical fields corresponding to the explanatory variables.
In the left slot, the numerical field corresponding to the dependent variable.
WARNING : In case of logistic regression with numerical response , the indicated field is used as "success" count, the next field is used as "failure" count.
Options
Option 1 : The summary of the regression.
Option 2 : The coefficients .
Option 3 : The supplementary results :
  1. Fitted values
  2. Predicted values
  3. Deviance residuals
  4. Pearson residuals
  5. Leverages (hat values)
  6. Cook's distances

Actions
Drag the first blue arrow to the numerical fields to create a new field whose values are the predicted values.
Drag the second blue arrow to the numerical fields to create a new field whose values are the errors (difference between predicted and true values)
In option 3: drag a blue arrow to the numerical zone to create a new field.
