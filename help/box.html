Definition
Für jede Kategorie der kategorialen Variablen werden einige Statistiken der quantitativen Variablen angezeigt:
  • Erstes Dezil
  • Erstes Quartil
  • Median
  • Drittes Quartil
  • Neuntes Dezil
Parameter
Oberes Feld: quantitative Variable(n), für die die Statistiken berechnet werden.
Linkes Feld: kategoriale Variable.
