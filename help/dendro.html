Definition
Hierarchical clustering of the dataset.
Parameters
In the slots, the numerical fields used to compute the proximity between the nodes.
Options
Option 1: Maximum (complete) linkage clustering.
Option 2: Minimum (single) linkage clustering.
Actions
Drag the special field "1" to a slot to add all the numerical fields.
Use the cursor to change the final number of classes.
Drag the pink arrow to the categorical fields to create a new field whose categories are the final classes of the dendrogram.
