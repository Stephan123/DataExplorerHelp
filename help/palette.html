Definition
Farbauswahl.
Aktionen
Ziehen Sie eine Farbe auf einen Graphen, um seine Grundfarbe zu ändern.
Die übrigen vom Graphen benötigten Farben werden ausgehend von dieser Grundfarbe so ausgewählt, dass sie auf dem Farbkreis gleich weit voneinander entfernt sind.
Beispiel: Falls nur zwei Farben benötigt werden (der Graph muss zwei Kategorien anzeigen), ist die zweite Farbe die Komplementärfarbe der Grundfarbe, z.B. Orange-Blau oder Grün-Rot.
