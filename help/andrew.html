Definition
Each data point `(x1,x2,x3,x4,x5...)` defines a finite Fourier series :
`x_1/sqrt(2) + x_2 sin(t) + x_3 cos(t) + x_4 sin(2t) + x_5 cos(2t) ... `
This function is then plotted between `-pi` and `+pi`
If there is structure in the data, it may be visible in the curves.

Parameters
In the right slots, the numerical fields used in the formula.
In the left slot, an optional categorical field used to color the lines.
