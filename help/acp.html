Definition
Convert a set of correlated variables into a set of uncorrelated principal components.
The represention of the data points in the cartesian coordinate system of the first two principal components separates at best the data points.

Parameters
In the right slots, the list of numerical fields to be part of the analysis.
In the bottom slot, an optional categorical field to color the points.
In the right slot, an optional categorical field used to label the points.
Options
Option 1: the data points in the plane of the first two principal components.
Option 2: the correlation circle in the plane of the first two components.
Option 3: the histogram of the eagenvalues, showing the proportion of the variance explained by each component.
Actions
Drag a numerical field to the graph to change its unit. The points will be represented as bubbles of size proportional to the numerical value.
