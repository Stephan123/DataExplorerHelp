Definition
Erstellt aus einer Menge korrelierter Variablen eine Menge unkorrelierter Hauptkomponenten.
Im Koordinatensystem werden die ersten beiden Hauptkomponenten dargestellt, die die Datenpunkte am besten voneinander separieren.

Parameter
Rechtes Feld: Liste der quantitativen Variablen, die in die Analyse eingehen.
Unteres Feld: optionale kategoriale Variable, nach der die Punkte eingefärbt werden.
Rechtes Feld: optionale kategoriale Variable, nach der die Punkte gelabelt werden.
Optionen
Option 1: Die Datenpunkte werden auf der Ebene angezeigt, die durch die ersten beiden Hauptkomponenten aufgespannt wird.
Option 2: Korrelationskreis auf der Ebene der ersten beiden Hauptkomponenten.
Option 3: Histogramm der Eigenwerte, welche angeben, welcher Varianzanteil durch die einzelnen Komponenten aufgeklärt wird.
Aktionen
Ziehen Sie eine quantitative Variable auf den Graphen. Die Punkte werden zu Kugeln, deren Größe den Wert dieser quantitativen Variablen darstellt.
