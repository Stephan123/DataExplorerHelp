Definition
Display the observations against later observations, or against the differences between the two.

Parameters
In the top slot, the qualitative field to be analyzed. The data must be sorted according to the time.

Options
Option 1 : `X_i` againts `X_(i-lag)`
Option 2 : `X_i` againts `X_i - X_(i-lag)`

Actions
Use the cursor to change the lag value.
