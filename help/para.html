Definition
A data point in m-dimensional space is represented by a polyline with vertices on the parallel axes.
The position of the vertex on the i-th axis correspond to the i-th coordinate of the data point.
Parameters
In the right slots, the numerical fields to be displayed.
In the left slot, an option categorical field used to color the lines.
Options
Option 0: Horizontal orientation. The variables are displayed from left to right, the values are displayed from bottom (minimum) to top (maximum).
Option 1: Vertical orientation. The variables are displayed from top to bottom, the values are displayed from left (minimum) to right (maximum).
