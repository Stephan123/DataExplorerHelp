Definition
Ein Datenpunkt im m-dimensionalen Raum wird als eine geknickte Linie dargestellt, deren Knicke auf parallelen Achsen liegen.
Der Knick auf der i-ten Achse entspricht dem Wert auf der i-ten quantitativen Variablen.
Parameter
Rechtes Feld: quantitative Variablen, die für die parallelen Achsen verwendet werden.
Linkes Feld: optionale kategoriale Variable, nach der die Linien eingefärbt werden.
Optionen
Option 1: Horizontale Anordnung. Die Variablen werden von links nach rechts angezeigt, die Werte von unten (niedrigster) nach oben (höchster).
Option 2: Vertikale Anordnung. Die Variablen werden von oben nach unten angezeigt, die Werte von links (niedrigster) nach rechts (höchster).
