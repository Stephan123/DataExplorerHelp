Definition
Barycentric plot of three variables which sum to a constant.
For each data point the three values are normalized so that their sum is 1.
Parameters
In the right slots, the numerical fields used in the analyze. If less than three fields, no analyze is produced. If more than three fields, the exceeding fields are ignored. Drag the special numerical field "1" to add all the fields.
In the left slot, an optional categorical field used to color the points.
In the right slot, an optional categorical field used to label the points.
Actions
Drag a numerical field to the graph to change its unit. The points will be represented as bubbles of size proportional to the numerical value.
