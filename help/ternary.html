Definition
Baryzentrischer Plot von drei Variablen, die sich zu einer Konstanten aufsummieren.
Für jeden Datenpunkt werden die Daten so normalisiert, dass ihre Summe 1 ist.
Parameter
Rechtes Feld: quantitative Variablen für die Analyse. Für weniger als drei Variablen wird keine Analyse durchgeführt. Bei mehr als drei Variablen werden die überschüssigen Variablen ignoriert. Ziehen Sie die spezielle quantitative Variable "1" in das Fenster, um alle quantitativen Variablen hinzuzufügen.
Linkes Feld: optionale kategoriale Variable, nach der die Punkte eingefärbt werden.
Rechtes Feld: optionale kategoriale Variable, nach der die Punkte gelabelt werden.
Aktionen
Ziehen Sie eine quantitative Variable hinein. Die Punkte werden zu Kugeln, deren Größe die Ausprägung der quantitativen Variablen entspricht.
