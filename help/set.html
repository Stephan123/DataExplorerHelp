Definition
Dieses Tool erstellt eine Dummyvariable aus den Kategorien einer kategorialen Variablen.
Parameter
Linkes Feld: Die kategoriale Variable, deren Kategorien verwendet werden.
Es können mehrere Felder gentuzt werden, um die Aufteilung nach und nach zu verfeinern.
Aktionen
Ziehen sie eine kategoriale Variable auf das Feld.
Klicken Sie auf den linken Knopf, um Kategorien aus der konstruierten Menge zu entfernen.
Klicken Sie auf den rechten Knopf, um der konstruierten Menge Kategorien hinzuzufügen.
Ziehen Sie den Pfeil auf die Liste kategorialer Variablen (oben rechts), um eine neu konstruierte Dummy-Variable mit den Werten YES und NO erstellen.
Zieheh Sie den Pfeil auf die Liste quantitativer Variablen, um eine neu konstruierte Dummy-Variable mit den Werten 1 und 0 erstellen.
