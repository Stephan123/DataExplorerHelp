Definition
Zwei Kategorien a und b von Variable 1 sind "assoziiert", falls es Beobachtungen (x, a) und (x, b) gibt, die derselben Kategorie x von Variable 2 angehören.
Parameter
Oberes Feld: kategoriale Variable, zwischen deren Kategorien Assoziationen dargestellt werden sollen.
Linkes Feld: kategoriale Variable, deren Kategorien die Assoziationen bestimmen.
Optionen
Option 1: Die Kategorien von Variable 1 werden entlang einer Linie in alphabetischer Reihenfolge angezeigt. Die Höhe des Bogens zwischen zwei Kategorien ist proportional zu der Anzahl von Assoziationen, die durch Variable 2 ausgelöst werden.

Option 2: Die Kategorien von Variable 1 werden auf einem Kreis angezeigt.
