Definition
Representation of data points in a polar coordinate system.
Parameters
In the top slot, the numerical field used as angular coordinate.
In the left slot, the numerical field used as radial coordinate.
In the bottom slot, an option categorical field used to color the points.
Options
Option 1: Representation of data as points.
Option 2: Representation of data as lines. If a categorical field is specified, as many lines as categories are displayed.
Actions
If the unit is "Radian", the angular coordinate is supposed to vary between 0 and 2π (6.28).
If the unit is "Degree", the angular coordinate is supposed to vary between 0 and 360.

Use the little handle to rotate the circle and position the origin.
