Definition
Eine Menge von Kategorien `{ cat_1, cat_2, ... cat_n }` von Feld 2 heißt solidarisch, falls es Beobachtungen `(x,cat_1)`, Beobachtungen `(x,cat_2)`, ...und Beobachtungen `(x,cat_n)` gibt, die in Feld 1 in eine gemeinsame Kategorie fallen.

Jede Zeile des Plot stellt eine Menge dar. Die roten Quadrate auf der linken Seite sind die solidarischen Kategorien von Feld 2. Die blauen Quadrate auf der rechten Seite sind die solidarischen Kategorien aus Feld 1, die eine Solidarität herstellen.

Die Mengen (Zeilen) sind nach absteigender Kardinalität geordnet.
Parameter
Oberes Feld: kategoriale Variable 1, die die Solidarität herstellen kann.
Linkes Feld: kategoriale Variable 2, deren Kategorien solidarisch sind.
